# Data and Code Availability

## Data Availability

This paper does not generate new empirical experimental data. The repository includes AISim-Cal synthetic simulation outputs, including CSV summaries in `simulation/outputs/` and generated figures in `figures/`. All cited external materials are public papers, official pages, or news reports listed in `references.bib`.

## Code Availability

The AISim-Cal synthetic simulation scripts, configuration files, generated CSV summaries, and generated figures are maintained with the preprint source. They are illustrative research artifacts for inspecting the calibration framework, not empirical forecasts of future scientific productivity.

## Repository Boundary

This public repository is a minimal source archive for manuscript inspection and synthetic simulation reproduction. Downloaded third-party PDFs, publisher supplementary files, saved HTML snapshots, and local citation-cache files are intentionally excluded.
